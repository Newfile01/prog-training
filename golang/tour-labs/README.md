# A Tour of Go path

This directory will follow this website lessons : [A Tour of Go](https://go.dev/tour/list)

## Variables

GOROOT : Compilator directory
GOPATH : Working directory

`go env`to show every settings.

GOTPATH should look like this :
├──── bin (executable commands)
├──── pkg (compiled version of sources)
└──── src (project sources and librairies)
